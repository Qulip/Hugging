# Hugging : 언택트 마음챙김 서비스

## 서비스 배경 💡

건강보험심사평가원 통계에 따르면 5년간 우울증 환자 수는 지속적인 증가 추세에 있습니다.
특히, 최근 코로나19의 확산으로 일상에 큰 변화가 생기면서 우울감과 무기력증이 늘었습니다.
이러한 이유들로, 지친 마음을 위로해주어 일상에 활기를 불어넣는 마음챙김 서비스를 생각하였습니다.
